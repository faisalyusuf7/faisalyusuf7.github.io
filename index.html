Featured Project

Pentagon Robot

Senior Design Project  ·  Aug 2025 – Present

An autonomous five-bar parallel planar robot designed for high-speed pick-and-place operations. Built to demonstrate advanced kinematic principles in an accessible educational platform.

≤ 5 sec cycle time
±0.5 mm accuracy
$327.80 / $350 budget
Dual NEMA 17 stepper motors
  • Led mechanical design and controls integration using a closed-loop five-bar linkage driven by dual base-mounted NEMA 17 stepper motors in a stacked direct-drive configuration
  • Applied Model-Based Design (MBD) in SolidWorks for full assembly, motion simulation, and workspace validation to eliminate link interference
  • Implemented inverse kinematics and randomised path planning on a Raspberry Pi 4; real-time motor actuation via Arduino UNO R3
  • Integrated suction-based end-effector with pressure-sensor feedback for reliable grasp confirmation
