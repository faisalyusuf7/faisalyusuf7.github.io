# faisalyusuf7.github.io

Personal portfolio  [faisalyusuf7.github.io](https://faisalyusuf7.github.io)

Built with HTML, CSS, and JavaScript.

## Adding media

Drop photos into `assets/images/` and videos into `assets/videos/`.
The `<img>` and `<video>` tags in `index.html` already point to the expected filenames.
